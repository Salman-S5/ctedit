# Custom made ncurses based terminal text editor written in C

## How to run
```
make run FILE="{file_name}.{ext}"
```
```
e.g. make run FILE="test.txt"
```
---

## Current Features:
- Cursor movement (arrow keys or hjkl)
- Functional viewport scrolling
- Commands:
  - CTRL+q: quit
  - CTRL+s: save file
